# Pizza-sales-report
This project presents a Power BI Sales Performance Dashboard for a fictional but realistic Pizza Company.   It provides insights into total sales, branch performance, manager contribution, Pizza type wise sales, and daily target achievement.

# Tools & Skills Used

Power BI Desktop

Power Query (ETL and cleaning)

Data Modeling

Bar/Donut/Map Visualizations

KPI Cards

Date Hierarchy & Filtering

Dashboard Design & Layout

# Key Insights

California and neighboring western states play a major role in sales.

Hot Pepperoni Feast is the revenue leader among pizza types.

Manager sales totals show consistent performance across the team.

Total sales of ₹189.48K suggest strong overall performance.

# Conclusion

This dashboard delivers a complete view of the pizza company’s sales operations. It enables managers to track performance, understand customer preferences, and optimize branch-level strategies.
